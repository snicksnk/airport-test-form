# Airport form
### Run project (with yarn)

```sh
$ yarn install
$ yarn start
```

### Run with docker

Install docker and docker-compose
- https://docs.docker.com/engine/installation/
- https://docs.docker.com/compose/install/


```sh
$ docker-compose up
```

### Build
yarn:
```sh
$ yarn build
```
docker-compose:
```sh
$ docker-compose exec frontend yarn build
```
